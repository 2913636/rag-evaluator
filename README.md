# RAG 质量自动评估器

> 用 LLM-as-Judge 评估 RAG 回答质量。三个维度：忠实度、相关性、完整性。

## 运行

```bash
pip install openai
set DEEPSEEK_KEY=你的Key
py evaluator.py
```

## 评估维度

| 维度 | 评估什么 | 典型扣分原因 |
|------|------|------|
| 忠实度 | 回答是否基于参考资料 | 编造数据、歪曲原文 |
| 相关性 | 是否回应了用户问题 | 答非所问、偏题 |
| 完整性 | 是否覆盖所有方面 | 遗漏关键信息 |

## 面试价值

- LLM-as-Judge 实战落地
- RAG 评测体系理解
- 结构化输出（JSON）能力
